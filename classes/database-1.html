Database 1

Course Overview

This course explored the architecture and utility of database management systems and progressed through reading/creating relational model diagrams to writing advanced SQL queries that utilize multi-table joins, aggregate functions, and sub-queries to transform raw data.

Skills Learned

  • Relational database design.
  • Proficiency in writing complex SELECT statements, including WHERE filters and ORDER BY sorting.
  • Merging datasets using UNION commands and managing data flow.
  • Implementing GROUP BY and HAVING clauses alongside aggregate functions.
  • Retrieving data across multiple tables using Inner and Outer Joins.
  • Designing nested queries (sub-queries) for advanced filtering and pre-calculations.

Content Covered

Data Modeling

Developing database diagrams, defining primary keys, and assigning accurate data types.

SQL Functions & Aggregation

Using SQL functions to calculate new information and manipulate data during retrieval.

Complex Joins

Combining data from three or more tables while maintaining query accuracy and performance.

System Management

Restoring database files and navigating database management interfaces.

Artifacts

Relational Database Diagrams

Determined field names and established logical relationships to develop a functional database diagram.

Inner Join Implementation

Identified the necessary components of a JOIN statement and developed queries using an INNER JOIN.

Aggregation & Grouping

Utilized the GROUP BY clause and aggregate SQL functions to calculate data.
