Data Security & Privacy

Course Overview

This course explores the legal, technical, and social frameworks necessary to protect sensitive information. It bridges high-level government regulations (GDPR, HIPAA, CCPA) with hands-on technical configurations required to secure databases, protect PII, and harden network infrastructure.

Skills Learned

  • Regulatory Compliance: Analyzing major data privacy laws and their impact on global business operations.
  • Advanced SQL Security: Managing logins, setting granular table permissions, and field-level encryption in Microsoft SQL Server.
  • Vulnerability Prevention: Utilizing stored procedures to encapsulate code and prevent SQL injection attacks.
  • Network Hardening: Configuring TCP/IP ports, Windows Defender Firewall, and password security protocols.
  • Defense Against Social Hacking: Identifying social engineering and phishing techniques to develop organizational defense strategies.
  • Incident Response: Developing formal breach response plans and professional incident notification protocols.

Content Covered

Data Classification & Access

Defining Personally Identifiable Information (PII) and utilizing the principle of “least privilege” to ensure users only access the data required for their specific roles.

Modern Security Threats

Exploring the role of employee training in preventing lapses and understanding how Artificial Intelligence (AI) and Machine Learning (ML) are evolving to both threaten and enhance modern data security.

Infrastructure & Monitoring

Understanding computer ports, monitoring application traffic, and maintaining a structured response plan to mitigate damage during security incidents.

Artifacts

Access Control & Restricted Transactions

Implementing database table permissions and restricted SQL logins to enforce security boundaries.

Incident Response Case Study

Drafting formal incident notification letters as part of a structured response plan following a data breach simulation.

Secure Code Encapsulation

Created stored procedures in SQL to enhance system performance while strictly controlling data access and preventing injection vulnerabilities.
